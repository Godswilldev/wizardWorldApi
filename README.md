  <h1 align="center">Wizard World API</h1>

## Description

Otugoh Backend Developer Challenge

## Prequisites

<ul>
  <li>MySql Database</li>
  
  <li>NodeJs</li>
</ul>

## Installation

```bash
 npm install
```

## Running the app

<ul>
  <li>After installation step, do the following</li>
  <li>create a .env file</li>
  <li>create a LOCAL_DATABASE_PASSWORD variable and the value should be your database password</li>
  <li>create a database named "wizardworld"</li>
</ul>

After the above steps run any of the commands below

```bash
# development
 npm run start

# watch mode
 npm run start:dev

# production mode
 npm run start:prod
```

## Test

```bash
# unit tests
 npm run test

# e2e tests
 npm run test:e2e

# test coverage
 npm run test:cov
```
